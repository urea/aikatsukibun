# AikatsuKibun プロジェクト規定 (.agent/GEMINI.md)

## 1. プロジェクト概要
- **名称**: AikatsuKibun (アイカツ！気分)
- **目的**: アイカツ！シリーズに関連した体験・情報の提供（詳細は後述の計画書参照）
- **優先言語**: 日本語（すべての対話・コードコメント・ドキュメント）

## 2. 開発・ドキュメント管理ルール
- **ディレクトリ構成の遵守**:
  - `App/`: コアソースコード
  - `Data/`: 非Git管理データ
  - `Dev/docs/`: ドキュメント（Artifacts含む）
  - `Dev/Tools/`: 開発補助ツール
- **ドキュメント命名規則**: `YYYYMMDD-HHMM_トピック名.md`
- **インデックス管理**: `Dev/docs/docs.md` が全ドキュメントの Source of Truth である。
- **セッション継承**: `walkthrough.md` や `current_status.md` は AI の自律判断で更新し、後継 AI への引き継ぎを常に行う。

## 3. デプロイとコミット
- **Gitコミット**: `git commit` は AI の判断で頻繁に実行する。
- **Push/デプロイ**: 本番公開に関わる `git push` はユーザーの明示的な許可を待つ。

## 4. 特記事項
- アイカツ！の世界観や意匠を尊重し、プレミアムなデザイン（リッチなCSS、マイクロアニメーション等）を追求する。
