## 1. プロジェクトの概要と目的
- **プロジェクト名**: AikatsuKibun (アイカツ！気分)
- **目的**: YouTube上の『アイカツ！』プレイ動画（硝子ドール等）に合わせて、画面上の3つのカラーボタン（赤・緑・黄）をタップして、気軽にゲームを遊んでいるような「気分」を楽しむスマートフォン向けWebアプリケーション。
- **基本設計要件**:
  - **画面分割比率（縦長スマホ画面前提）**: 
    - 上部 10%: ヘッダー（タイトル: アイカツ！気分）
    - 中央 30%: YouTube動画表示エリア（硝子ドールプレイ動画 `fYibOFCMpnE` を埋め込み）
    - 下部 40%: タップ遊びエリア（薄いピンク背景）
    - 下部 20%: 空白エリア
  - **タップ機能**: 三角形に配置された赤（上）、緑（左下）、黄（右下）の3つのボタン。タップ時（またはPCキーボード/マウスクリック時）にCSSアニメーションによって「強く光る」フィードバックのみを提供（スコア計算などはなし）。

---

## 2. 現在の進捗状況と完了済み機能
基本実装はすでに **すべて完了** しています。

### 実装済みの機能
1. **UIレイアウトの再現 (`App/index.html`, `App/style.css`)**
   - `100dvh`（Dynamic Viewport Height）と Flexbox を用いて、スマートフォン画面に最適化された 10% / 30% / 40% / 20% の比率レイアウト。
   - 直径 `110px` の大きな丸型ボタン（赤・緑・黄）を中央タップエリアに三角形配置。
2. **プレミアムな発光エフェクト (`App/style.css`)**
   - `.active` クラス付与時に、外側の大きなグロー（`box-shadow`）と内側のホワイトグロー（`inset`）、さらに `scale(0.95)` によるプッシュフィードバックを実行。
3. **レスポンシブな動画埋め込み (`App/index.html`)**
   - YouTube埋め込み iframe により、指定の『硝子ドール(SPステージ) プレイ動画』が中央エリアいっぱいに表示されるよう実装済み。
4. **マルチデバイス入力制御 (`App/main.js`)**
   - スマホのタッチ操作 (`touchstart`, `touchend`, `touchcancel`) への対応（スクロール抑止のための `preventDefault()` を設定し、極めて高い応答速度を実現）。
   - PCでのマウス操作 (`mousedown`, `mouseup`, `mouseleave`) に対応。
   - PCキーボード操作対応: **`W`（赤・上） / `A`（緑・左下） / `D`（黄・右下）** キーで連打・ホールドに対応。
