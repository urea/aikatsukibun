# AikatsuKibun (アイカツ！気分) パワーアップアイデア提案書 (20260520-0738)

本ドキュメントは、「アイカツ！気分」Webアプリケーションが持つ、アーケード筐体を疑似体験する「気分と快感」をさらに高め、ユーザー体験を極大化するための具体的なパワーアップアイデアを提案するものです。

---

## 🌟 1. ビジュアル＆演出のプレミアム化（きらめきの強化）

### ① スペシャルアピール（全画面カットイン）演出の再現
- **概要**: プレイ中（タップ中）に特定のコンボ数に達した際、またはプレイヤーが「ここぞ！」というタイミングで発動できる必殺技「スペシャルアピール」演出を実装。
- **演出詳細**:
  - 画面全体にきらびやかなゴールドレースや羽が舞い散るCSSアニメーション。
  - キャラクターのカードが飛び出すような3Dライクなフリップカットイン。
- **効果**: 単調なタップ作業にドラマチックな山場を作り、ライブ感を最高潮に高めます。

### ② DCD筐体風LED・ネオンフレームの実装
- **概要**: YouTube動画を囲む「プレミアムフレーム」をさらに進化。
- **演出詳細**:
  - 実際のデータカードダス（DCD）筐体のように、音やタップのアクションに合わせて、フレームの「ネオン管（LEDライン）」が虹色にグラデーション発光・点滅するアニメーション。
  - グラスモルフィズムと組み合わせることで、サイバーかつファンタジーな筐体の美しさをブラウザ上に再現します。

## 🎮 2. ゲーム性と操作性の拡張（「気分」から「ゲーム」へ）

### ① スコア＆判定システム（Perfect / Good / Miss）
- **概要**: 現在はリズムに合わせて自由にボタンを叩くだけですが、簡易的な判定・コンボ機能を実装。
- **演出詳細**:
  - ボタンを叩いた位置に「PERFECT!」「GOOD!」といったカラフルできらめくポップアップテキストを表示。
  - タップごとに「COMBO 10」のように数字が跳ねるカウンターを表示し、Missをするとコンボが途切れる仕組み。
  - *注：正確な音ゲー譜面を作らなくても、YouTubeのBPMに合わせて簡易的にタイミング判定を生成するか、ユーザーの打鍵感そのものを評価する仕組みから段階的に導入可能です。*

### ② キーボード・マッピングのカスタマイズ機能
- **概要**: 現在は `W` / `A` / `D` に固定されているPC用キーボードマッピングを変更可能に。
- **演出詳細**:
  - 設定パネルから、自分の叩きやすいキー（例：左手＝`S`、上＝`Space`、右手＝`L` など）を自由に割り当てられるオプションを追加し、LocalStorageに保存。

## 🎴 3. コレクション要素とアイドルランク（継続モチベーションの向上）

### ① バーチャル「アイカツ！カード」排出システム
- **概要**: 1曲プレイし終わる（または一定のタップ数に達する）と、筐体からカードが排出されるワクワク感をシミュレート。
- **演出詳細**:
  - プレイ完了時に「カード排出アニメーション」が走り、ランダムでオリジナル（またはインスパイアされた）デジタルアイカツ！カードを獲得。
  - 獲得したカードは「カードアルバム（図鑑）」に永続保存。
  - 特定のドレスカードを「マイコーデ」にセットすると、プレイ中のボタンのジュエルカラーや、飛び散るパーティクルの色・形状が変化する着せ替え要素。

### ② ファン獲得と「アイドルランク」システム
- **概要**: プレイごとにスコアやタップ数に応じて「ファン（Fan）」を獲得。
- **演出詳細**:
  - ファン数が一定値に達すると、「レッスン生」→「デビュー」→「トップアイドル」へとアイドルランクが上昇。
  - ランクアップ時に専用のきらきらした王冠やバッジのビジュアルが解放される。

## 📱 4. UXと共有機能のスマート化

### ① カスタムプレイリストのURL共有（SNS連携）
- **概要**: 自分で「カスタム動画追加」したリストやお気に入りの曲順を、他のユーザーに簡単に共有できる機能。
- **演出詳細**:
  - お気に入りリストのデータ（動画IDやタイトル）をBase64などで圧縮エンコードし、`?playlist=xxxx` という共有用URLを発行。
  - このURLをX（旧Twitter）などでシェアすれば、他のファンも同じプレイリストで即座に遊ぶことができます。

### ② PWA（プログレッシブWebアプリ）化による完全モバイル最適化
- **概要**: スマホのホーム画面にアプリアイコンを追加し、全画面（ブラウザのURLバーなどを非表示）で遊べるようにする。
- **演出詳細**:
  - アドレスバーやフッターのブラウザUIが完全に消えるため、没入感が実機アプリと同等になります。
  - 音声アセット（効果音）や基本UIスタイルをキャッシュすることで、電波の悪い場所でもタップSEが遅延なく発音可能になります。
