# AikatsuKibun (アイカツ！気分) 超進化実装計画 (20260519-1410)

本ドキュメントは、既存の「アイカツ！気分」Webアプリケーションを、アイカツ！の持つプレミアムで輝かしい世界観を極限まで再現した「超リッチな体験型アプリケーション」へと進化させるための具体的な実装計画案です。

---

## 1. 超進化の3大コアコンセプト

### 🌟 コンセプトA：心躍るプレミアム・ビジュアル＆マイクロアニメーション
現在のシンプルなピンクと白のUIから、アイカツ！特有の「星屑（スターダスト）」「ゴールドレース」「ジュエル（宝石）」の意匠を取り入れたプレミアムなデザインへと刷新します。
- **グラスモルフィズム（Glassmorphism）**: 操作パネルやヘッダーにガラスのような透明感と美しさを与えます。
- **筐体風ボタンデザイン**: 3色のボタンをただの丸型から「ゴールドの縁取り」「ジュエルカット調グラデーション」に変更。
- **キラキラ・パーティクルエフェクト**: ボタンをタップした瞬間に、画面上に無数のきらめく星（スターパーティクル）が動的に生成され、ふわっと消えていく演出をJavaScriptで実装します。
- **微細な背景アニメーション**: 背景にオーラのような淡い光のグラデーションや、星がゆっくりと流れるアニメーションを常時裏で走らせ、アプリ全体が「生きている」ような躍動感を与えます。

### 🎵 コンセプトB：ダイナミックな「楽曲選択（プレイリスト）」と「テーマチェンジ」
硝子ドール固定だった仕様を進化させ、アイカツ！の主要な名曲をセレクトできる機能を追加します。さらに、**選択した楽曲のタイプ（キュート、クール、セクシー、ポップ）に合わせてアプリ全体のUIテーマ（配色や背景アニメーション）がリアルタイムに切り替わる**システムを構築します。

| 楽曲名 | タイプ | UIテーマの演出 | YouTube動画ID |
| :--- | :--- | :--- | :--- |
| **硝子ドール** | クール / セクシー | ゴシックパープル＆ダークブラック、怪しく光る薔薇と星屑 | `fYibOFCMpnE` |
| **アイドル活動！** | キュート | ハッピーピンク＆イエロー、ハートと星が飛び交う明るいポップ | `1_m0kP8yD6Y` |
| **ダイヤモンドハッピー** | ポップ | ビタミンオレンジ＆スカイブルー、弾けるダイヤと元気な星 | `H0MhA6n_qj4` |

### 🔊 コンセプトC：Web Audio APIによる「筐体風タップ効果音（SE）」と振動
- **タップSEの導入**: ボタンを叩いた際に、本物のゲーム筐体を叩いているような心地よい打鍵音や「シャンッ」「キラリーン」といったきらびやかな効果音をWeb Audio APIで発音させます。（※ブラウザのポリシーに基づき、最初の画面タップ時にオーディオ再生を許可するインタラクティブなUIを挟みます）
- **触覚フィードバック（バイブレーション）**: モバイル端末でのタップ時、心地よい微細な振動（Vibration API）を発生させ、タップの快感を最大化します。

## 2. 画面レイアウトの進化イメージ
既存の「10% / 30% / 40% / 20%」の比率レイアウトを維持しながら、要素の中身を極限まで豪華にします。

- **ヘッダー（10%）**: 
  - 高級感あるゴールドとジュエルの装飾。Google Fonts（`Outfit` や `Inter`）を導入し、きらきらと輝くテキストシャドウを適用。
  - 左端に「楽曲セレクト」のプルダウン（またはグラスモルフィズムなセレクトボタン）を配置。
- **YouTube動画エリア（30%）**:
  - iframeの周囲に、テレビの枠のようなゴールドのプレミアムフレームをあしらい、動画自体がステージのモニターのように見える演出。
- **タップ遊びエリア（40% + 20%の一部）**:
  - 背景を星空グラデーションに変更。
  - 三角形に並ぶ3つの筐体風ジュエルボタン。
  - タップ時に飛び散る「スターパーティクル」。
- **下部エリア（20%）**:
  - コーデ選択カード（簡易的な観賞用アイカツ！カード）などを美しく並べるギャラリースペース、または「Audience（観客）」のサイリウムが揺れるアニメーションを追加。

## 3. 具体的な開発フェーズとスケジュール

### Phase 1: ビジュアル＆テーマ基盤の刷新（CSSの極大進化）
- Google Fonts (`Outfit`) のインポートと、アイカツ！専用カラー変数（Cute-Pink, Cool-Purple, Pop-Yellow）の定義。
- 背景グラデーションアニメーションと、ジュエルカット風ボタンのCSS実装。
- 楽曲セレクターのHTML/CSS追加。

### Phase 2: 動的なテーマ変更＆パーティクルエフェクトの実装（JavaScriptの進化）
- 楽曲セレクターの変更を検知し、埋め込みYouTube動画のIDと、`#app` のクラス（`theme-cute`, `theme-cool`, `theme-pop`）を動的に切り替えるロジックの実装。
- ボタンタップ時に、マウスクリック/タッチ位置からCSSアニメーションで飛び散る「星屑スパンコール（DOM要素）」を生成・消滅させるパーティクル関数の実装。

### Phase 3: 音響・振動と最終調整（UXの極大進化）
- Web Audio API を用いたシンセサイズタップ音（または軽量なオーディオ素材）の発音実装。
- スマホ向け Vibration API の組み込み。
- ローカル環境での入念なレスポンシブ動作テスト。
