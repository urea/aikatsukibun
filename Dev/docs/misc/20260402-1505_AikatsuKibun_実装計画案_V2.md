# AikatsuKibun (アイカツ！気分) 実装計画 V2 (20260402-1505)

## 概要
提供いただいた仕様書（`202604021450_AikatsuKibun.md`）に基づき、スマートフォン縦画面サイズ（ブラウザ幅に合わせる形）にて、上部にタイトル、中央に指定のYouTube動画、下部に三角形に配置された3つのボタン（赤、緑、黄）を配置したWebアプリケーションを構築します。スコア計算は行わず、タップ時の視覚的なフィードバック（発光）のみを提供します。

## ユーザーレビュー依頼事項
> [!IMPORTANT]
> 前回の反省を踏まえ、以下の手順で進めて良いか**必ず**ご確認をお願いいたします。
> 1. **Vite環境の再構築**: 前回の途切れたプロセスをやり直し、`App/` ディレクトリ配下に Vite (Vanilla JS) 環境をクリーンな状態で構築します。
> 2. **レイアウト実装 (CSS)**: `vh` (Viewport Height) と CSS Grid/Flexbox を使用し、指定の割合（タイトル10%, 動画30%, タップ40%, 空白20%）を厳密に再現します。
> 3. **動画埋め込み**: YouTube IFrame APIを使用、または単なる `<iframe>` 埋め込みを利用して指定動画 (`fYibOFCMpnE`) を表示します。
> 4. **ボタン実装**: 赤（上）、緑（左下）、黄（右下）の三角形配置を行い、タップ/クリック検知時に CSS アニメーションで「光る」エフェクトを付けます。

- **YouTubeの自動再生について**: ブラウザの制限により、ユーザーの操作（タップ等）がないと動画の自動再生や音声再生がブロックされる場合があります。動画の再生開始はYouTube標準の埋め込みプレイヤーの再生ボタンを押して始める想定でよろしいでしょうか。
