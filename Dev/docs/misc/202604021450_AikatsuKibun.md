## 概要
- アイカツ！の楽曲を再生する。。
- スコアなどは表示しない。あくまで動画に合わせてボタンを叩くことを楽しむだけ。

## 画面イメージ
- スマートフォンの縦長画面を基本とする

- 画面全体
  - 上部：タイトル　上部10%
  - 中央：YOUTUBE動画表示エリア　中央30%
  - 下部：タップエリア　下部40%
  - 空白エリア：下部20%

- YOUTUBE動画表示エリア
  - youtubeの動画を最大幅で表示し、縦のサイズは動画サイズにあわせる
- タップエリア
 -　三角形にならんだ3つのボタン
 - タップで光る
```
　赤
緑　黄
```
